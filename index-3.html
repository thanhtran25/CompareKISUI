Flow CU — Tung buoc

Kha nang nhieu call
①
Tinh fetchSize
fetchSize = max(limit × 2, 3). Voi limit=10 → fetchSize=20. Qua nho voi dataset lon.
final int fetchSize = Math.max(limit * 2, 3); // = 20
↓
②
Bat dau vong lap while(true)
offset=0, tiep tuc fetch tung batch nho, tang offset sau moi lan.
while (true) { var contracts = fetch(accountId, from, to, offset, fetchSize); if (contracts.isEmpty()) break; // filter... offset += fetchSize; }
↓
③
Filter overdue trong tung batch
Moi contract duoc kiem tra matureDate < today va status [A,P,F]. Ket qua tich luy dan.
↓
④
Kiem tra dieu kien dung vong lap
Dung khi co du item cho page VA batch vua fetch la batch cuoi (it hon fetchSize). Rat kho xac dinh som.
if ((pageResults.size() >= limit) && (contracts.size() < fetchSize)) break;
↓
⑤
Van de: Co the fetch qua nhieu
Neu du lieu overdue nam cuoi dataset, vong lap phai duyet qua toan bo. Voi 10,000 contracts → co the 500+ lan call TTL.

1️⃣ totalCount ≤ 100 (chi 1 call)
Khi TTL tra ve totalCount = 60

remaining = 60 − 100 = −40 ≤ 0

→ Bo qua Call #2 hoan toan
→ allContracts = res1.list() (60 items)
→ Chi toc do 1 lan call TTL
2️⃣ totalCount = null tu API
Mot so truong hop TTL co the tra null cho totalCount

long totalCount = res1.totalCount() != null ? res1.totalCount() : 0;

→ totalCount = 0
→ remaining = 0 − 100 = −100 ≤ 0
→ Bo qua Call #2, dung du lieu call #1
3️⃣ start vuot qua tong overdue
User truyen start=100 nhung chi co 40 overdue

from = Math.min(100, 40) = 40
to = Math.min(110, 40) = 40

→ subList(40, 40) = list rong
→ Tra ve empty list, totalOverdue=40
4️⃣ res1.list() hoac res2.list() = null
TTL co the tra null thay vi empty list

new ArrayList<>(res1.list() != null ? res1.list() : List.of())

Tuong tu cho res2:
if (res2.list() != null) allContracts.addAll(res2.list());

→ Khong NullPointerException
5️⃣ Khong co contract nao overdue
Sau khi filter, overdueList.size() = 0

→ totalOverdue = 0
→ from = min(start, 0) = 0
→ to = min(start+limit, 0) = 0
→ subList(0, 0) = empty list

Tra ve response voi totalOverdue=0, list=[]
6️⃣ TTL tra loi (errorCode != success)
Sau moi call TTL deu co validate:
ttlClient.validateErrorCode(res.errorCode());

→ Neu loi: nem TtlUnauthorizedException hoac TtlFailureException
→ Xu ly o tang cao hon (controller/filter)
→ Khong xu ly mot nua roi tra data sai
